This website will show you step by step how to make an FTP server and an HTTP server.

How to make an FTP server
  1. First, you need to download FileZilla Server here.
  2. Once FileZilla Server is installed, go to settings and make sure the server listens to port 21.
  3. Now go to Passive Mode Settings and enter your IP address and check the custom port range and enter ports 50000 to 53000.
  4. Now go to Admin Interface Settings and enter a password for your server.
  5. Now click OK and you're done.
  6. Now go to Edit and click Users
  7. Now go to shared folders.
  8. In the Users section click Add
  9. Now enter the name of the user
  10. Then add a directory
  11. Click OK and you're done with the server

    Now you have to connect a client to the server to show that it works.

  2. After you have it installed, type in the Host IP, Username, and password
